# RTX 4080 baselines

This page is populated from real benchmark results in `benchmarks/results/`.
If you see a "not run" row, the configuration was never executed (e.g. because
the model was gated or the OOM happened before the run finished). We never
fill these with synthetic numbers.

To reproduce: install with `[train]` extras and run

```bash
canifinetune bench --model <model_id> --method <lora|qlora> --seq-len <N> --micro-batch-size <M> --steps 3
canifinetune report --benchmarks benchmarks/results --out docs/rtx4080_baselines.md
```

The repository ships with results collected on:

- GPU: NVIDIA GeForce RTX 4080 (16 GB)
- CUDA driver: 13.2 (current at time of run)
- OS: Windows 11
- PyTorch: see each result's `env` field

A baseline summary (auto-generated by `canifinetune compare`) lives at
`benchmarks/results/_compare.md` after running the bench suite.

See `scripts/run_4080_baselines.sh` for the exact commands used.
